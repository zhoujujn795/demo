# Docs Index

本目录用于承载 AI 股票分析面板的实现文档。当前阶段不写代码，只把需求拆成 Codex 后续可执行的任务说明。

## 文档列表

- [01-product-requirements.md](./01-product-requirements.md): 产品目标、用户流程、功能边界。
- [02-technical-architecture.md](./02-technical-architecture.md): 推荐技术栈、项目结构、模块职责。
- [03-api-and-data-contracts.md](./03-api-and-data-contracts.md): 后端接口、数据结构、Supabase 表设计。
- [04-codex-implementation-tasks.md](./04-codex-implementation-tasks.md): 按阶段拆分的 Codex 实现任务。
- [05-acceptance-checklist.md](./05-acceptance-checklist.md): MVP 验收标准、测试点、部署检查。

## 使用方式

后续开发时，优先按 `04-codex-implementation-tasks.md` 的阶段顺序推进。每完成一个阶段，再用 `05-acceptance-checklist.md` 做检查，避免遗漏面试项目要求。
